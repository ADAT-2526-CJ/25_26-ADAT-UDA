Ejercicio 10 _ Cifrado y Descifrado con el algoritmo de Vigenère.    
Alumno/a: Urtzi Diaz Arberas
Unidad Didáctica: UD1 - Python
 
Este programa permite cifrar y descifrar textos usando el algoritmo clásico
de Vigenère. Se trabaja únicamente con letras A-Z (mayúsculas). Los espacios
y caracteres especiales se eliminan para simplificar.
El usuario puede elegir entre cifrar o descifrar un texto, proporcionando
una clave. La clave se repite o recorta según sea necesario para coincidir
con la longitud del texto.
